EDI UPS

.NET Framework / HTML / CSS / Javascript

Sistema desenvolvido para integrar e gerenciar comunicações via EDI (Electronic Data Interchange) com a transportadora UPS. A aplicação permite o envio e recebimento automatizado de documentos eletrônicos, como pedidos, faturas e notificações de envio, garantindo maior agilidade e rastreabilidade nas operações logísticas. A interface web foi construída com HTML, CSS e JavaScript, enquanto o backend utiliza .NET Framework para processamento e integração com APIs e serviços da UPS.
