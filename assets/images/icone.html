👇 Como usar no seu HTML

Dentro do seu .logo, coloque assim:

html
<div class="logo">
  <svg width="48" height="48" viewBox="0 0 48 48" xmlns="http://www.w3.org/2000/svg">
    <rect width="48" height="48" rx="12" fill="#1E1E1E"/>
    <text x="50%" y="50%" text-anchor="middle" dominant-baseline="middle" fill="#FFFFFF" font-family="Arial, sans-serif" font-size="18" font-weight="bold">
      RN
    </text>
  </svg>
</div>

Se ainda não estiver aparecendo:

  1. Verifique se o CSS não está escondendo .logo ou .header.

  2. Tente aumentar o tamanho do SVG temporariamente pra ver se aparece (width="100").

  3. Veja se não há conflitos de z-index ou display: none.

Se quiser, posso aplicar um estilo mais moderno, tipo um círculo com sombra, ou uma animação suave. É só dizer o estilo que você curte!
